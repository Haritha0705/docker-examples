# Docker Examples

This repo contains examples of Docker usage:

1. **Single Dockerfile App**  
   Folder: `single-dockerfile`  
   Simple Node.js app running in a single container.

2. **Multi-Container App with Docker Compose**  
   Folder: `multi-docker-compose`  
   Frontend + Backend app demonstrating multiple containers working together.
